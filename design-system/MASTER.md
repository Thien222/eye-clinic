# EyeClinic Pro — Design System (UI UX Pro Max)

**Target:** Medical Clinic / Healthcare Management  
**Pattern:** Data-Dense Dashboard + Task Workflow  
**Style:** Soft UI Evolution + Accessible & Ethical  

## Colors

| Token | Hex | Usage |
|-------|-----|-------|
| Primary | `#0D9488` | CTAs, active nav, links |
| Primary Light | `#CCFBF1` | Hover backgrounds |
| Secondary | `#0891B2` | Secondary actions |
| Background | `#F8FAFC` | App shell |
| Surface | `#FFFFFF` | Cards, panels |
| Text | `#0F172A` | Headings, body |
| Muted | `#64748B` | Labels, captions |
| Success | `#059669` | Completed, connected |
| Warning | `#D97706` | Waiting, caution |
| Danger | `#DC2626` | Delete, offline |
| Border | `#E2E8F0` | Dividers |

## Typography

- **Font:** Plus Jakarta Sans (Google Fonts)
- **Headings:** 600–700 weight
- **Body:** 400–500 weight
- **Minimum touch target:** 44px

## Components

- Soft shadow: `shadow-soft` (0 1px 3px rgba(15,23,42,0.06))
- Radius: `rounded-xl` cards, `rounded-lg` inputs/buttons
- Transitions: 200ms ease
- Icons: Lucide only (no emoji icons)

## Anti-Patterns (Healthcare)

- No AI purple/pink gradients
- No neon colors
- No emojis as UI icons
- High contrast text (4.5:1 minimum)

## Status Colors

| Status | Style |
|--------|-------|
| Waiting | Amber |
| Processing | Teal |
| Completed | Emerald |
| Billing | Sky |
